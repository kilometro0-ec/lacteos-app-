# lacteos-app-
# Sistema de Control de Ventas de Lácteos

## Objetivo
Registrar ventas, clientes y control de stock usando Google Sheets.

## Funcionalidades
- Registro de ventas
- Registro de clientes nuevos
- Control de inventario
- Actualización automática de stock

## Herramientas
- Google Sheets
- GitHub

## Flujo
1. Se registra cliente
2. Se registra venta
3. Se descuenta stock automáticamente
